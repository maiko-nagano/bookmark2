## 課題内容（どんな作品か）

- サイト名・URL・コメントを入力して送信ボタンを押すと、データベースに登録され、登録内容を一覧で表示する
- 編集ボタンを押すと編集画面に飛び、登録された内容を変更して更新ボタンを押すとデータベースが更新される
- 一覧画面で削除ボタンを押すと「本当に削除しますか？」という確認メッセージが出て、OKとするとデータベースが削除される
